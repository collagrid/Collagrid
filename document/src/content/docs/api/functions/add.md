> **add**(`a`, `b`): `number`

Adds two numbers together.

## Parameters

• **a**: `number`

The first number.

• **b**: `number`

The second number.

## Returns

`number`

The sum of the two numbers.

## Example

```typescript
const result = add(2, 3);
console.log(result); // 5
```
